# 專案名稱：線上算命系統 (Online Fortune-Telling System)

## 1. 專案概述
- **背景與動機**：在現代快節奏的生活中，許多人面對壓力與迷惘時，渴望尋求心靈的慰藉與方向。然而，親自前往廟宇或算命館可能受限於時間與地點。因此，打造一個線上算命系統，讓使用者能隨時隨地獲得指引與心靈寄託。
- **目標用戶**：對未來感到迷惘、需要尋求心靈指引的人，以及對命理、星座有興趣，習慣線上操作的年輕族群或一般大眾。
- **核心價值主張**：提供便捷、多樣化且具備高度隱私的線上算命服務，結合抽籤與捐款功能，打造全方位的數位心靈寄託平台。

## 2. 功能需求

以下為系統的主要功能：

1. **會員帳號管理**
   - 作為 **使用者**，我希望 **能註冊並登入系統**，以便 **追蹤和管理我的個人資料與所有的算命紀錄**。
2. **算命、占卜、抽籤**
   - 作為 **訪客或會員**，我希望 **能使用線上算命、塔羅占卜或廟宇抽籤等功能**，以便 **獲得針對當下問題的指引與解析**。
3. **儲存算命結果**
   - 作為 **會員**，我希望 **系統能自動儲存我每次的算命與抽籤結果**，以便 **我日後隨時登入回顧與驗證**。
4. **捐香油錢（線上捐獻）**
   - 作為 **使用者**，我希望 **能透過線上支付方式捐獻香油錢**，以便 **表達敬意、祈求平安，並支持平台的營運**。
5. **每日運勢與社群分享**
   - 作為 **使用者**，我希望 **能獲得專屬的每日運勢，並將結果分享至社群平台**，以便 **作為每日參考，並與親朋好友互動**。

## 3. 非功能需求
- **技術限制**：後端採用 Python Flask 框架，前端使用 HTML/CSS 與 Jinja2 模板引擎進行渲染，資料庫採用輕量級的 SQLite。
- **效能與安全考量**：
  - 保障個人隱私：使用者的出生年月日等敏感算命個資必須受到妥善保護，密碼應進行雜湊加密（如 bcrypt）。
  - 金流安全：「捐香油錢」功能若介接真實第三方支付（如 Line Pay/綠界），需確保交易過程皆透過 HTTPS 加密傳輸。
  - 系統穩定度：確保系統能流暢運行並具備防重放（CSRF protection），防止惡意多次提交抽籤請求。

## 4. MVP 範圍 (Minimum Viable Product)

- **Must Have (必須有)**：
  - 會員註冊與登入。
  - 一種基礎的抽籤/算命機制（例如：隨機抽取六十甲子籤並顯示籤詩與解籤）。
  - 會員儀表板：能夠瀏覽過去的歷史算命儲存結果。
- **Should Have (應該有)**：
  - 捐香油錢的模擬或真實串接功能。
  - 基本的 RWD 響應式介面，支援手機與電腦瀏覽。
- **Nice to Have (加分項)**：
  - 各種不同種類的占卜（如塔羅、八字排盤）。
  - 算命結果的社群一鍵分享按鈕，或產生分享精美圖片。
